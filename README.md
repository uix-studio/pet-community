# 멍냥멍냥 - 반려동물 커뮤니티

Next.js 14 + React Query + Tailwind 기반 반려동물 커뮤니티 모바일웹.

## 스택

- Next.js 14, TypeScript, Tailwind CSS
- React Query (TanStack Query)
- Lucide React (아이콘)

## 로컬 실행

```bash
npm install
./run.sh
# 또는: npm run build && PORT=3300 npm run start
```

→ http://localhost:3300

## 배포

Vercel에서 자동 빌드/배포 지원. `DEPLOY.md` 참고.
